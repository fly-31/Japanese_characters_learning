# Main idea

The website will help Korean speakers to learn Japanese characters by identifying the Korean pronounciation (and maybe meaning too) of Japanese words

# Features

- Only pronounciation (발음시험): Given words such as まぶしい, you only input the korean pronounciation (in this case, 마부시이). You may locally store words (>50 words for each category). The user will also be able to adjust how many words to be tested on, with input setting for each word.
- pronounciation and definition: There will be two inputs. First is for entering pronounciation and the second will be for entering meaning in Korean. Same structure as only pronounciation mod except the two input part.
- Seperate 히라가나, 가타카나, and 한자 into 3 options. 
- Button for moving to index.html in character folder. Then also add button in character to make it transferrable both direction
- Use the same main page format as the character website.
