# Main idea

The website will help Korean speakers to learn Japanese characters by identifying the Korean pronounciation given a Japanese character

# Features

- From n to n: The user will pick from any number to another number that will be used to select how many 히라가나 or 가타카나 characters they will practice with. For example, 1 to 20 will pick from first character to 20th character. The order of characters will be based on 아, 카 , 타... 파 행. For example, first 6 히라가나 will be あいうえおか. However, when giving to the user this order will be randomized and then the user can input korean pronounciation in order. For example, if the user selects 1 to 6 the website could give えかうおあい, as randomly mixed in order, and the user can input 에카우오아이 as the correct answer.
